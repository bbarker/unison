We should be able to `match` on empty types like `Void`.

```unison
Void.absurdly : '{e} Void ->{e} a
Void.absurdly v = match !v with
```

```unison
Void.absurdly : Void -> a
Void.absurdly v = match v with
```

And empty `cases` should also work.

```unison
Void.absurdly : Void -> a
Void.absurdly = cases
```

But empty function bodies are not allowed.

```unison:error
Void.absurd : Void -> a
Void.absurd x =
```
